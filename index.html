Rafter

Backbone.js, Coffeescript, Rails backend

Point of sale system for bookstores to quickly process textbook rentals. Built with backbone.js using coffeescript and REST API requests to a rails backend. Bookstores needed a much simpler and faster checkout flow than bookrenter.com and required some additional prompts. We created an iOS app that incorporates the barcode scanning and card reading capabilities of a Linea Pro and deployed several mobile checkout stations to 5 different campus bookstores.

Unfortunately, this app is only available to college bookstores, but if for some extremely unlikely reason you are a college bookstore, sign up here!

Phelps School

Design, Wordpress, HAML, SCSS, JavaScript

The Phelps School is an all boy's boarding school located outside of Philadelphia. They sought an easy-to-update website, with options for tall, bold images, and mobile accessibility. I went with wordpress for content management, and hooked it up to use Sass and Haml. The site is fully responsive, and uses Off Canvas Navigation at the smallest breakpoint.

Presence iPhone app

Interface Design, Usability Testing

Presence is another breakable toy started while at thoughtbot. The client (a developer) wanted a tool to help train him to be more mindful and in the moment. The application periodically pings the user, asking them if they're 'present'. It then adjusts the variable timeframe window for the next reminder, according to the user's input. The statistics view shows progress, both as binary results and as sparklines for trends.
